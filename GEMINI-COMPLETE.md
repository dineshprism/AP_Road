## **🚀 Quick Start Guide:**

### **Step 1: Get Gemini API Key**
1. Visit [Google AI Studio](https://aistudio.google.com/app/apikey)
2. Create new API key
3. Copy the key

### **Step 2: Configure System**
```bash
# Set up environment
cd server
cp .env.example .env
# Add your API key: GEMINI_API_KEY=your_key_here

# Install dependencies
npm install @google/generative-ai

# Build and start
npm run build
npm start
```

### **Step 3: Test System**
```bash
# Open test suite
start gemini-rag-test.html

# Test all features:
# 1. Backend Health Check
# 2. Single Accident Analysis  
# 3. Batch Accident Analysis
```

### **Step 4: Use Application**
```bash
# Start frontend
npm run dev

# Open: http://localhost:8080
# Login → View Past Submissions → Click "Analyse"
```

## **🔧 Troubleshooting:**

### **Common Issues & Solutions:**

#### **"Invalid Gemini API key"**
- ✅ Check `.env` file for correct API key
- ✅ Verify key is active in Google AI Studio
- ✅ Ensure key has sufficient quota

#### **"Backend not responding"**
- ✅ Start backend: `cd server && npm start`
- ✅ Check port 3000: `netstat -an | findstr :3000`
- ✅ Verify build: `npm run build`

#### **"Analysis Failed"**
- ✅ Check browser console (F12) for errors
- ✅ Verify submission data exists
- ✅ Test with `gemini-rag-test.html`
