## **🔍 Troubleshooting Guide:**

### **If Backend Not Responding:**
1. Check if port 3000 is free: `netstat -an | findstr :3000`
2. Kill Node processes: `taskkill /F /IM node.exe /T`
3. Restart backend: `cd server && npm start`

### **If Ollama Not Working:**
1. Check if port 11434 is free: `netstat -an | findstr :11434`
2. Restart Ollama: `ollama serve`
3. Check model: `ollama list`

### **If AI Analysis Fails:**
1. Check browser console (F12) for errors
2. Verify authentication token exists
3. Test with different optimization levels
4. Check backend logs for detailed error messages
